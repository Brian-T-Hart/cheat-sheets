# Minifier

## Usage

Minify a JavaScript or HTML file using Node.js and Terser.

```shell
node minify.js <input.js> [output.min.js]
```

- `<input.js>`: Path to the JavaScript file you want to minify.
- `[output.min.js]`: (Optional) Path for the minified output file. Defaults to `output.min.js`.

## Example

```shell
node minify.js ../test.js ../test.min.js
```

## Setup

1. Run `npm install` in the `Minify` folder to install dependencies.
2. Use the command above to minify files.
